# GitOps Portfolio Starter Template

## Objective
Demonstrate GitOps workflow using a personal portfolio project with Git-based version control, branching, and PRs.

## Features
- Git branching and merging
- Pull request workflow
- Clean and descriptive commit history
- Personal portfolio starter UI
- GitOps-style change managemnet using GitHub

## Setup
```bash
git clone <https://github.com/ArpitMetkar26/gitops-portfolio>
cd gitops-portfolio
open src/index.html
